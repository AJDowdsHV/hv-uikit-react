# Grid

Examples of the Grid System proposed by Design System using our Container and Grid components. The implemented Grid is based in the [Material UI](https://v3.material-ui.com/layout/grid/#grid) implementation.

The definitions where set following the Design System directives:

| Breakpoint | Width (in px) | Gutters (in px) | Number of columns |
| ---------- | ------------- | --------------- | ----------------- |
| xs         | [0-600[       | 15              | 4                 |
| sm         | [600-960[     | 15              | 8                 |
| md         | [960-1270[    | 30              | 12                |
| lg         | [1270-1920[   | 30              | 12                |
| xl         | [1920-...[    | 30              | 12                |
