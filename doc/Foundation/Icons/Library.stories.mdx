# Icon library

You can import our React icons from the icon library as follows (example for the `ActualSize` icon):

```js
import { ActualSize } from "@hv/uikit-react-icons/dist";
```

And to use them:

```jsx
<ActualSize />
```

This will be the result:

# Pictograms library

You can import our React pictograms from the icon library as follows (example for the `FemaleRunning` icon):

```js
import { FemaleRunning } from "@hv/uikit-react-icons/dist/pictograms";
```

And to use them:

```jsx
<FemaleRunning />
```
